# MediaPipe Face Mesh：FACEMESH_* 連線與索引參考

本表由 **Google MediaPipe** 官方 `face_mesh_connections.py` 自動推導（見下方來源），用於對照「某索引出現在哪些語意連線集合裡」。**這不是解剖學命名表**；語意僅來自常數名稱（如 `LEFT_EYE`）。

## 來源與更新方式

- 上游檔案（Apache-2.0）：[`mediapipe/python/solutions/face_mesh_connections.py`](https://github.com/google-ai-edge/mediapipe/blob/master/mediapipe/python/solutions/face_mesh_connections.py)
- 本專案副本：`tools/mediapipe_face_mesh_connections.py`
- 重新產生本文件：在專案根目錄執行 `python tools/generate_facemesh_index_reference.py`
