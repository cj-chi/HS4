# 臉部對齊：基本邏輯

當**兩張臉足夠像**時，下面這條就是我們做「滑桿 ↔ landmark」對齊的基本邏輯。

---

## 一句話

**Slider（或 Slider 組合）在正面投影、並做完中心與比例調整後，必然與對應的 landmark 的位置／線段／比例／距離重合。**

- **比例對比例**、**線段對線段**、**距離對距離**：投影到正面 2D 之後，兩邊的量應該一致。
- 有些差異會來自 **深度（3D）**，但**正面投影看不到深度**，所以在純正面 2D 下，我們對齊的就是：比例↔比例、線段↔線段、距離↔距離。

---

## 展開說

1. **對齊條件**：兩張臉「足夠像」（同一人、或幾何上很接近）。
2. **座標一致**：把「遊戲臉」正面投影到螢幕後，先做**中心對齊**與**比例（縮放）對齊**，和「照片／landmark 所在的那張圖」對齊到同一個 2D 座標系。
3. **對應關係**：
   - 某個 **Slider**，或 **多個 Slider 組出來的某條線段、某個距離、某個比例**，在正面投影後，就對應到 landmark 的**某條線段、某個距離、某個比例**。
   - **比例對比例**、**線段對線段**、**距離對距離**：投影後兩邊一致。
4. **深度誤差**：真實 3D 會有深度，所以 3D 上可能有一點誤差；但在**正面投影**時，深度不會被看見，所以我們在 2D 上對齊時，可以不考慮深度造成的誤差，直接要求「投影後的量 = landmark 的量」。

---

## 對我們實作的含義

- **從 landmark 推 Slider**：用螢幕上的臉部 landmark（位置、線段、距離、比例）算出對應的幾何量，再對應到 Slider（或 Slider 組合）。
- **驗證**：遊戲臉正面投影＋中心與比例對齊後，同一組 Slider 產生的線段／距離／比例，應與該畫面上的 landmark 線段／距離／比例重合。
- **比例對比例、線段對線段、距離對距離**：我們選的 ratio、segment、distance 都要在「投影並對齊後」與 landmark 一致；深度誤差在正面 2D 下可忽略。

---

## 解題方法（針對單一 Slider）

要精確、快速得到對的設定值，可以照下面步驟做（**以 source code 與遊戲資源為準**，以下為方法論；具體 x 的定義需從原始碼／曲線解析得出）：

### Step 1：針對某個 Slider，研究原始碼

- 找出：**正面投影後**，有沒有一個**要素 x** 幾乎完全由這個 Slider 控制。  
- 例如：遊戲人物**下巴的曲線**，主要由**下巴寬度**（某個 index）控制。

### Step 2：用同一幾何量 x 從 Landmark 取得

- 確定 **x** 的**幾何定義**（例如「一條曲線兩端在同一水平 0，最低點與 0 的垂直差距」）。  
- 到 **Landmark** 上找：**哪些 landmark 可以組成「同樣定義的 x」**（同一條曲線、同一個距離、同一個比例）。  
- 在 landmark 上**量測同一個 x**（同一比值／同一距離／同一曲線量），得到 **x(landmark)**。

### Step 3：對應到 Source Code 裡能呈現同一 x 的設定

- 在**原始碼或遊戲資源**（list/customshape、ShapeAnime 曲線）裡，找到：**哪個 Slider 值（或曲線上的 rate）會讓遊戲正面投影後產生同樣的 x**。  
- 也就是：**x(landmark) = x(game; slider_value)** → 反推 **slider_value**。  
- 若遊戲是「slider → 曲線 → x」，就需透過**曲線表／查表／calibration**，從 x 反推 slider；這樣才能**精確、快速**得到對的設定值。

### 舉例（純說明，實際以 source code 為準）

- 假設從 source code 得知：**x** = 「一條曲線兩端固定在同一水平 0 時，曲線最低點與 0 的**垂直差距**」（或該差距／臉高的比值）。  
- 那我們就應在 **Landmark** 上量**同一個定義**：例如 FACE_OVAL 下巴段，兩端同高、取最低點與該水平線的距離（或除以臉高成比例）。  
- 再在 source code／曲線表中，找到**能呈現同樣 x（或同樣比值）的 Slider 設定**，把量到的 x(landmark) 對過去，得到 slider 值。

這樣做，才能讓「遊戲正面投影的 x」與「landmark 的 x」對齊，並精確反推 Slider。
