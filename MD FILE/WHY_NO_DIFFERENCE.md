# 為什麼看不出差別？

## 原因：參數沒有寫進角色卡

目前流程只做兩件事：

1. **預覽圖**：可選換成全白（`--white-preview`），所以卡片**縮圖**是白的。
2. **Trailing（ChaFile）**：和基底卡**完全一樣**，沒有把 `eye_size=95`、`eye_span=95` 等寫進去。

所以進遊戲讀 **big_eyes_white.png** 時，角色臉型、身體和讀 **AI_191856.png** 是同一份資料，**不會有差別**。

---

## 參數存在哪裡？

只存在旁邊的 **big_eyes_white.params.json** 裡，例如：

```json
{"mapped_params":{"eye_span":95,"eye_size":95,...},"source_image":"..."}
```

遊戲不會讀這個 JSON，所以**不會自動套用**。

---

## 要怎麼看到「眼睛變大」？

### 方式一：手動在遊戲裡調（現在就能做）

1. 在 HS2 讀取 **big_eyes_white.png**。
2. 進角色編輯，找「眼睛大小」「眼距」等滑桿。
3. 把對應滑桿調到 **95**（或 JSON 裡寫的數值）。

這樣就會看到差別；差別來自你手動調的滑桿，不是卡片檔案被我們改過。

### 方式二：之後實作「寫回 ChaFile」

要讓**存檔裡就帶大眼睛**，需要：

- 解析 ChaFile 的 Parameter 區塊（MessagePack）。
- 把我們的 `eye_size` / `eye_span` 對應到遊戲的欄位（例如 `eyeWidth`、`eyeHeight`、`eyeSpacing` 等）。
- 改寫該區塊後再寫回 PNG trailing。

目前還沒做這一步，所以**卡片內容完全沒被改**，自然看不出差別。

---

## 總結

| 項目           | 有沒有變 |
|----------------|----------|
| 卡片縮圖       | 有（可選全白） |
| 卡片裡的臉型參數 | **沒有** |
| 遊戲中看到的臉   | 和基底卡一樣，**沒有差別** |

要「看得出差別」：要麼在遊戲裡依 JSON **手動調滑桿**，要麼之後實作**把參數寫回 ChaFile**。
